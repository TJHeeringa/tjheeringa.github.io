How to define: the natural numbers

Author

T.J. Heeringa

Published

February 18, 2025

This is the first post in a series called “How to define …?”. In this series, we will answer the question “How is X actually defined?”. We will answer this for various math concepts we learn about in middle school, so that at the end of the series one knows exactly how most of the math learned in middle school is defined.

Is knowing how these things is defined of any use in everyday life? No, not really. However, it does provide a solid fundament for any aspiring mathematician, and perhaps - more importantly - it just a series of fun facts to learn.

The natural numbers

We start with the natural numbers. These numbers are the first numbers learned at school. They are the counting numbers and zero. So, \(0\), \(1\), and \(2\) are natural numbers but \(1,234,567,890\) is too.

To be able to define these numbers and not get stuck in circular reasonings or end up with paradoxes, we have to be careful with where we start. We will start with sets. This is the common modern approach.

The basic building block: the set

So, what is a set? A set is collection of different things. These things can be anything: they can be numbers, lines, chairs, banana’s or even other sets. The things in the set are called elements. We write sets using braces and separate all the elements by comma’s. For example, \(\{1,2\}\) is the set with the numbers \(1\) and \(2\) as elements. If a particular thing is an element of a set, we use the \(\in\) symbol (pronounced: in) and if is it not we use the \(\not\in\) symbol (pronounced: not in). For example, \(1\in\{1,2\}\) but \(3\not\in\{1,2\}\).

We do not care about how these elements are ordered in the set. Hence, we say the set \(\{1,2\}\) is the same as the set \(\{2,1\}\). We also only care about unique elements, so the set \(\{1,2,1\}\) is the same set as the set \(\{1,2\}\).

Set-builder notation

There are many sets. Some of these are special, and in many cases we are interested in sets constructed from others. It helps to have some convenient way of writing that down. We use set-builder notation for that. You have already seen the braces for the sets and the \(\in\) for being an element. We need a few more of these to be able to write down and understand the definition of the natural numbers.

  1. Emtpy set \(\emptyset\): The empty set is the set without anything in it, i.e. \(\emptyset=\{\}\).
  2. Subset \(\subseteq\): If all elements of set \(T\) are also elements of set \(S\), we write \(T\subseteq S\) and say that \(T\) is a subset of \(S\).
  3. Union \(\cap\): The set \(S\cap T\) is the union of the set \(S\) and the set \(T\). It contains all the elements of the set \(S\) and the set \(T\).
  4. Intersection \(\cup\): The set \(S\cup T\) is the intersection of the set \(S\) and the set \(T\). It contains all the elements that are in both the set \(S\) and the set \(T\).
  5. Restriction \(\{ x\in S \; | \; \phi(x) \}\): The set \(\{ x\in S \; | \; \phi(x) \}\) contains the elements from the set \(S\) that satisfy the condition \(\phi\). E.g. if \(S=\{1,\text{`T-rex'}, \odot\}\) and \(\phi(x) =\) “\(x\) is a a number”, then \(T = \{ x\in S \; | \; x\text{ is a number} \} = \{1\}\).

Defining sets, but more rigorous

In our description of sets, we said that sets are collections of different things. To prevent the circular reasonings or end up with paradoxes, we need to make sure that we add some extra rules to what we say is acceptable as sets. After this, anything that doesn’t satisfies this rules will no longer be called a set or part of the sets. The list here includes the rules from the description above and the extra ones.

  1. Axiom of the empty set: The exists a set with no element in it \(\emptyset\), called the empty set.
  2. Axiom of extensionality: If two sets have the same elements, then they are the same set.
  3. Axiom of union: For any two sets, the union is a set.
  4. Axiom schema of specification: Any restriction of a set is only a set when the condition which we use to restrict the set is purely based on the elements of the unrestricted set.
  5. Axiom of pairing: For any two sets, there exists a set with these sets as elements.
  6. Axiom of extensionality: For any set \(S\), there is a set which contains all subsets of the set. This set is written as \(\mathcal{P}(S)\) and is called the power set of the set \(S\).
  7. Axiom of regularity: Every set \(S\), that is not the equal to the empty set, has an element \(x\) such that \(x\) and \(S\) have no elements in common.
  8. Axiom of schema of replacement: If any function that transforms certain sets into other sets is applied to each element of a set, then the result is a set.
  9. Axiom of choice: For every set consisting of sets which are not empty, there is a way to pick one element from each non-empty set.
  10. Axiom of infinity: There exists a set which has the empty set as element and the property that if \(x\) is an element of the set then the set \(x\cup{x}\) is also an element of the set.

The first 4 rules are effectively what we described before. The next 2 rules are expansion to what we described before, but do not introduce new symbols. The seventh rule excludes some (problematic) cases. The eight and nineth is not strictly necessary for ours goals, but are an essential part of advanced mathematics. The last one makes sure that infinitely large sets exists, and is specially designed for defining the natural numbers in a unique way.

The fourth rule is effectively a constraint on the conditions \(\phi\) that we can use. This is to avoid Russell’s paradox. This paradox describes the set \[ R = \{ x \;|\; x\not\in x \} \] This set is the set of all sets that are not elements of themselves. If \(R\) is not an element of \(R\), then it must be an an element of \(R\). If \(R\) is an element of itself, then it cannot be an element of itself. This is contradictory, so such sets should be excluded. We can also use this rule to write the intersection and the empty set: If \(S\) and \(T\) are sets, then \[ S\cap T = \{ x\in S \;|\; x\in T\} \] and \[ \emptyset = \{ x\in S \; |\; x\not\in S\} \] We have included the first one for convenience: we avoid the technicalities of defining the symbol for the empty set otherwise, and we use it in rule 7 and 10 already.

The (problematic) cases that the seventh one excludes are the following three:

  1. A set cannot be an element of itself. If \(T=\{S\}\) for a set \(S\), then \(T\) is not the empty set. It has \(S\) as only element. So, according to the rule, \(S\) and \(T=\{S\}\) should have no elements in common; \(S\cap \{S\}=\emptyset\). This is the same as saying the \(S\) cannot be an element of \(S\).
  2. For two sets \(S\) and \(T\), only \(S\) can be an element of \(T\) or \(T\) can be an element of \(S\). Both cannot be true. According to rule 5, \({S,T}\) is a set. According to this rule, either \(S\) and \({S,T}\) should have no elements in common or \(T\) and \({S,T}\) should have no elements in common. If \(S\) is an element of \(T\), then \(S\) and \({S,T}\) have an element in common. The same holds if \(T\) is an element of \(S\).
  3. There cannot be loops or infinite descending sequences of sets. We can write such a sequence as \(x_1\ni x_2\ni x_3\ldots\), which goes on forever with sets \(x_k\) with \(k\) a natural number. Since each \(x_k\) is a set, we can use rule 5 to conclude that there must be a set \(S\) with all these sets \(x_k\), i.e. \(S=\{x_1,x_2,x_3,\ldots\}\). However, there is not a \(x_k\) that shares no elements with \(S\); \(x_1\) shares the set \(x_2\) with \(S\), \(x_2\) shares the set \(x_3\) with \(S\), etc.

By excluding these cases, we get that every set can be constructed using the the empty set and power sets. We start the set \(V_0\), which is simply the empty set \(\emptyset\). The next set \(V_k\) is the powerset of the previous set, i.e \(V_k=\mathcal{P}(V_{k-1})\). The first few set \(V_k\) are \[ \begin{align} V_0 &= \emptyset \\ V_1 &= \mathcal{P}(V_0) = \{\emptyset\} \\ V_2 &= \mathcal{P}(V_1) = \{\emptyset, \{\emptyset\}\} \\ V_3 &= \mathcal{P}(V_2) = \{ \emptyset, \{\emptyset\}, \{\{\emptyset\}\}, \{\emptyset, \{\emptyset\}\}\} \end{align} \] For every set \(x\), there is a set \(V_k\) such that \(x\) is an element of \(V_k\).

Rule 10 says there exists a particular set which satisfies the given properties. This does not imply that there can only be one set that satisfies these properties. In fact, there are infinitly many such sets.

From sets to natural numbers

To go from ours definition of sets towards the natural numbers, we need to answer two questions

  1. How is a natural number defined in terms of sets?
  2. Does the set of all natural numbers actually exist?

Question 1: How is a natural number defined in terms of sets?

The natural numbers can be expressed as sets using the Von Nuemann ordinals. In this construction,

The answer to first question is that zero is the empty set and all other numbers are the set of all the numbers before them. Hence, the natural numbers are the sets \[ \begin{align} 0 &= \emptyset \\ 1 &= \{ 0 \} = \{\emptyset\} \\ 2 &= \{ 0, 1\} = \{\emptyset, \{\emptyset\}\} \\ 3 &= \{ 0, 1, 2\} = \{\emptyset, \{\emptyset\}, \{\emptyset, \{\emptyset\}\} \} \\ 4 &= \{ 0, 1, 2, 3\} = \{\emptyset, \{\emptyset\}, \{\emptyset, \{\emptyset\}\}, \{\emptyset, \{\emptyset\}, \{\emptyset, \{\emptyset\}\}\}\} \end{align} \] and so on. This makes sure that the number of elements in the set matches the number. More importantly, it makes sure the numbers correspond to going from \(x\) to \(x\cup{x}\), since \[ \begin{align} 1 &= \{ 0 \} = 0 \cup \{0\} \\ 2 &= \{ 0,1 \} = \{0\} \cup \{1\} = 1 \cup \{1\} \\ 3 &= \{ 0,1,2 \} = \{0,1\} \cup \{2\} = 2 \cup \{2\} \\ 4 &= \{ 0,1,2,3 \} = \{0,1,2\} \cup \{3\} = 3 \cup \{3\} \end{align} \] and so on. We will use this to answer the second question.

The second question asks whether the \[ \mathbb{N} = \{ 0, 1, 2, 3, 4, \ldots \} \] is a set. Whilst the notation suggests this is true, but the rules don’t tell us directly that it is true. Luckily, we can derive from them that it does.

Let’s use the letter \(I\) for a set that satisfies the requirements of rule 10. Every natural number is an element of \(I\). From the rule, we immediately see that \(0\in I\). From the rule, it follows that \(0\cup\set{0}\) must be an element of \(I\) too. This is nothing else than saying that \(1\in I\), since we showed that \(0\cup\set{0}\) is just a different way of writing the set \(1\). We can repeat this argument for every natural number to show that they are all elements of \(I\).

Now, there can be additional elements in \(I\) that are not natural numbers. We need to show that after removing those additional elements we still have a set. Since every set satisfying rule 10 contains the natural numbers, the intersection of all these should too. This intersection is written as \[ \omega = \{ x\in I \;|\; x\in S\text{ for all subsets $S$ of $I$ that satify rule 10}\} \] By rule 4, this is a set. This set satisfies rule 10: - \(\emptyset\) is in every subset \(S\), so also in \(\omega\). - If \(x\in \omega\), then \(x\in S\) for every subset \(S\). This implies that \(x\cup\set{x}\) is in every \(S\) too. Hence, \(x\cup\set{x}\) is an element of \(\omega\). Does this match \(\mathbb{N}\)? Well, suppose it is not. This means that there exists an element \(y\) in for each \(S\) that is not a natural number and for which there is no element \(x\) in for each \(S\) such that \(y=x\cup\set{x}\) satisfying rule 10. However, the set \[ \{ x \in S \; | \; x\neq y\} \] also satisfies rule 10. This means there is a set \(S\) without \(y\), which is contradictory. Hence, \(\omega\) must be equal to \(\mathbb{N}\).

Conclusion

In summary, natural numbers are specific sets constructed from the empty set, and the formal definition of the set of natural numbers \(\mathbb{N}\) is given by \[ \mathbb{N} = \{ x\in I \;|\; x\in S\text{ for all subsets $S$ of $I$ that satify rule 10}\} \] Working out this definition gives us \[ \mathbb{N} = \{0, 1, 2 ,3, \ldots\} \] which is exactly as expected.
