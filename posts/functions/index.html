How to define: functions

Author

T.J. Heeringa

Published

March 1, 2025

Addition and multiplication are functions on the natural numbers. To properly define them, we need to define what a function is in the first place, and we need to show that the functions we call “addition” and “multiplication” actually exist based on the rules we are using. For the latter, we are going to use the recursion theorem. So, in this post, we are going to define functions and prove this recursion theorem is true.

The definition of functions

When we look at addition and multiplication in an abstract sense, we can interpret them as something that takes 2 natural numbers and gives us one back. Things take that a certain number of elements and give us one back are called functions.

Ordered sets

We see that for a function a distinction needs to be made between what is goes into it and what goes out of it. However, our framework is based on sets and sets do not have an order for their elements. Whilst it is possible to define ordered sets in full generality within our frameworks, we only need ordered pairs.

Definition 1. Let \(a\) and \(b\) be sets. The ordered pair \((a,b)\) is the set \(\{\{a\},\{a,b\}\}\).

That there is an order stems from that \[ (a,b) = \{\{a\},\{a,b\}\} \neq \{\{b\},\{a,b\}\} = (b,a) \] This allows us to think of \(a\) as the first coordinate and \(b\) as the second coordinate of the pair \((a,b)\). Later, we will use the first coordinate for what goes into the function and the second coordinate for what comes out of the function.

Cartesian products of sets

To be able to properly define a function, we need be able to talk about all the ordered pairs with the first coordinate coming from one set and the second coordinate from some (maybe different) set. The set of all ordered pairs with \(a\) in a set \(A\) and \(b\) in the set \(B\) is the Cartesian product of \(A\) and \(B\). For its definition, recall that the power set \(\mathcal{P}(A)\) is the set of all subsets of the set \(A\).

Definition 2. Let \(A\) and \(B\) be sets. The Carthesian product of \(A\) and \(B\) is \[ A\times B = \{ z\in \mathcal{P}(\mathcal{P}(A\cup B)) \;|\; z=(a,b)\text{ for some }a\in A, b\in B \} \]

Functions

A function is a subset of a Carthesian product with an extra requirement.

Definition 3. A function \(f\) from \(A\) to \(B\) is a subset of the Carthesian product \(A\times B\) for which for every \(a\in A\) there exists a unique \(b\in B\) such that \((a,b)\in f\).

If we are given an element \(a\in A\), then we write \(f(a)\) for the element of \(B\) such that \((a,f(a))\in f\). We say that \(f\) maps an element \(a\) to \(f(a)\). We call \(A\) the domain of \(f\), \(B\) the codomain of \(f\), and \[ f(A) = \{ f(a) \; | \; a\in A \} \] the range of \(f\). Moreover, the notation \[ f:A\to B,\; a \mapsto f(a) \] is shorthand for stating that \(f\) is a function from \(A\) to \(B\) that \(a\) to \(f(a)\), and we shorten this to \(f:A\to B\) when we are only interested in or only know the domain \(A\) and codomain \(B\).

Functions can have several properties. We will list three important ones here, so that the definitions of these are easy to find. However, we will not use them immediately. Other important ones, like continuity, we will introduce when we need them.

Definitions 4,5,6. Let \(f:A\to B\).

  • \(f\) is injective, when \(f(x)=f(y)\) implies that \(x=y\) for all \(x,y\in A\).
  • \(f\) is surjective, when for every \(b\in B\) there exists an \(a\in A\) such that \(f(a)=b\).
  • \(f\) is bijective, when \(f\) is injective and surjective.

The recursion theorem

When we define a function \(f:A \to B\), we have to specify the domain \(A\), the codomain \(B\), and how \(f\) maps elements of the domain to its range. This function is only well-defined when the subset of \(A\times B\) it describes actually exists and satisfies the uniqueness requirement. Sometimes this is easy to check, sometimes it is not. For function on the natural numbers \(\mathbb{N}\), a particular convenient theorem is the recursion theorem.

The successor function

The recursion theorem is expressed in term of the successor function \(S:\mathbb{N}\to\mathbb{N}\). The natural numbers were defined as \[ \begin{align} 1 &= \{ 0 \} = 0 \cup \{0\} \\ 2 &= \{ 0,1 \} = \{0\} \cup \{1\} = 1 \cup \{1\} \\ 3 &= \{ 0,1,2 \} = \{0,1\} \cup \{2\} = 2 \cup \{2\} \\ 4 &= \{ 0,1,2,3 \} = \{0,1,2\} \cup \{3\} = 3 \cup \{3\} \end{align} \] and so on. Here, we repeatatly use the expression \(x\cup \{x\}\). The successor function \[ S:\mathbb{N}\to\mathbb{N}, \; x \mapsto x\cup \{x\} \] allows us to write this more compactly, because with this successor function \(S\) we can write \[ \begin{align} 1 &= S(0) \\ 2 &= S(1) \\ 3 &= S(2) \\ 4 &= S(3) \end{align} \tag{1}\] and so on.

The term element \(S(n)\) is called the successor of \(n\), due to its interpretation as mapping \(n\) to the next natural number.

Statement and proof

Intuitively, the recursion theorem says that we can repeat Equation 1 with a different function and a different starting point. This creates for a function \(f:X\to X\) with domain and codomain the same set \(X\) and a starting point \(a\in X\) the elements \[ \begin{align} a_0 &= a \\ a_1 &= f(a) \\ a_2 &= f(a_1) \\ a_3 &= f(a_2) \\ a_4 &= f(a_3) \end{align} \] so that \(a_4=f(f(f(a)))\). According to the theorem, these elements describe a function \[ u: \mathbb{N} \to X,\; n\mapsto a_n \] Functions like this are super usefull, and we will use them to define addition and multiplication.

Let’s now phrase the theorem and prove it.

Theorem 1 Let \(X\) be a set. If \(f:X\to X\) and \(a\in X\), then there exists a function \(u:\mathbb{N}\to X\) such that \(u(0)=a\) and \(u(S(n))=f(u(n))\) for all \(n\in\mathbb{N}\).

Proof. Let \[ \mathcal{G} = \{ A\subseteq \mathbb{N}\times X \;|\; (0,a)\in A\text{ and }\forall (n,x)\in A: (n,x)\in A \implies (S(n),f(x))\in A \} \] Here, we use the shorthand notation \(\forall\) for the words `for all', the shorthand notation \(:\) for the words `such that', and the shorthand notation \(\implies\) for the word `implies'. Thus, this is the set of all subsets \(A\) from the Carthesian product \(\mathbb{N}\times X\) that include the ordered set \((0,a)\) and whenever \((n,x)\) is an element of \(A\) then \((S(n),f(x))\) must also be an element of \(A\). Clearly, this list is not empty since \(\mathbb{N}\times X\) is an element of \(\mathcal{G}\)

Consider the special element \(u\) of \(\mathcal{G}\) defined by \[ u = \{ (n,x)\in \mathbb{N}\times X \; |\; (n,x)\in A\text{ for all }A\subseteq \mathcal{G}\} \tag{2}\] By construction, \(u\) is the smallest subset of \(\mathcal{G}\).

We are going to show that this special element \(u\) is the function that the theorem says exists. To this end, let \[ \mathcal{U} = \{ n\in \mathbb{N} \; | \; \text{there exists a unique }x\in X\text{ such that }(n,x)\in u \} \] \(u\) is a function if and only if \(\mathcal{U}=\mathbb{N}\). We are going to show that \(\mathcal{U}=\mathbb{N}\) by contradiction. For the contradiction, we consider the cases \(0\not\in \mathcal{U}\) and \(S(n)\not\in\mathcal{U}\) for \(n\in\mathbb{N}\).

Suppose \(0\not\in\mathcal{U}\). This implies that there exists some \(b\in X\) with \(b\neq a\) such that both \((0,a)\in u\) and \((0,b)\in u\). We see that \((0,a)\in u\backslash \{(0,b)\}\). We also see that \((S(n),f(x))u\backslash \{(0,b)\}\) for all \((S(n),f(x))u\), since \(S(n)\neq 0\). Combined this shows that set \(u\backslash \{(0,b)\}\) must also be an element of \(\mathcal{G}\). However, Equation 2 shows that \(u\subseteq A\) for all \(A\in\mathcal{G}\). So, in particular, we must have that \(u\subseteq u\backslash \{(0,b)\}\). This is not possible, so our initial assumption must be false. Thus, \(0\in \mathcal{U}\).

Suppose \(S(n)\not\in \mathcal{U}\) for some \(n\neq 0\). This implies that there exists some \(y\in X\) with \(y\neq f(x)\) such that both \((S(n),f(x))\in u\) and \((S(n),y)\in u\). Clearly, \(S(n)\neq 0\). Thus, \((0,a)\in u\backslash \{(S(n),y)\}\). Now suppose that \((m,z)\in u\backslash \{(S(n),y)\}\).

  • If \(n=m\), then \(z=x\) and \((S(m),f(t))=(S(n),f(x))\in u\backslash \{(S(n),y)\}\).
  • If \(n\neq m\), then \(S(n)\neq S(m)\) and \((S(m),f(t))\in u\backslash \{(S(n),y)\}\)

In both cases, \((S(m),f(t))\in u\backslash \{(S(n),y)\}\). Hence \(u\backslash \{(S(n),y)\}\in \mathcal{G}\). By the same arguments as before, we must have that $uu{(S(n),y)}. This is not possible, so our initial assumption must be false. Thus, \(S(n)\in \mathcal{U}\).

Hence, \(\mathcal{U}=\mathbb{N}\). This means \(u\) is a function with domain \(\mathbb{N}\) and codomain \(X\), and it satisfies the requirements, since \(u\in \mathcal{G}\).
